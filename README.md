Penjelasan ACT Pert1-Pert6



Pertemuan 1

Activity berisi dasar-dasar pemrograman Java dan struktur kode awal yang digunakan sebagai pondasi untuk pertemuan selanjutnya.



Pertemuan 2

Activity membahas konsep OOP (Object-Oriented Programming), khususnya inheritance dan overriding. Program terdiri dari class User, Buyer, Seller, dan Admin yang menunjukkan pewarisan dan perilaku berbeda pada masing-masing subclass.



Pertemuan 3

Pertemuan 4

Activity mengembangkan aplikasi CRUD berbasis GUI menggunakan Java Swing dan Hibernate. Program mencakup view form mahasiswa, controller sebagai pengatur logika, model dan tabel model, serta konfigurasi Hibernate untuk koneksi ke database.



Pertemuan 5

Pertemuan 6

Activity menggabungkan Spring Boot dengan Java Swing. Data mahasiswa dapat diinput lewat tampilan GUI, kemudian diproses melalui controller, service, repository, dan disimpan ke database. Data ditampilkan kembali melalui tabel GUI.
